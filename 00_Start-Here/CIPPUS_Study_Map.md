# CIPP/US Study Map (Portfolio Companion)

## Core concepts to master
- US privacy model: sectoral laws + comprehensive state privacy laws (patchwork)
- Key principles: notice/transparency, purpose limitation, minimization, retention, security safeguards
- Processor/service provider obligations and contracting posture
- Consumer/contractual rights readiness (delete/access/correct/export)
- Incident/breach coordination and notification expectations
- Vendor oversight: subprocessors, flow-down obligations, change notification
