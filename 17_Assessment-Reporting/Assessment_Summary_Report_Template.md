# Vendor Assessment Summary Report (Template)
## Executive summary
Decision + tier + residual risk + conditions.

## Scope
Data types/classification, access paths, subprocessors.

## Risk analysis
Inherent score drivers, key risks, mitigating controls.

## Evidence summary
Key EV items + confidence + freshness.

## Residual risk + decision
Decision matrix application and rationale.

## Remediation plan
Owners, deadlines, verification evidence.
