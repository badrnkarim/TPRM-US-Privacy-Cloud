# Audit Trail Checklist (TPRM)
Auditor evidence expectations:
- Tiering + scoring rationale
- DDQ results + evidence index
- Decision record + approvals
- Contract/privacy annex evidence
- Remediation closure proof
- Monitoring logs + reassessment triggers
