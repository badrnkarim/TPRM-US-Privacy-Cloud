# Ongoing Monitoring Playbook

## Monitoring signals
- Security incidents/breach news and disclosures
- SOC2/ISO renewal status (High/Critical)
- Major product changes affecting access/data flows
- Subprocessor changes (High/Critical)
- Increased access privileges requested
- Repeated remediation slippage

## Triggers for reassessment
- New data types or higher classification
- New privileged/network access path
- Vendor incident affecting confidentiality/integrity/availability
- Contract renewal for High/Critical

## Cadence
- Critical: continuous monitoring + annual reassessment
- High: annual reassessment
- Medium: 24 months
- Low: 36 months or major change
