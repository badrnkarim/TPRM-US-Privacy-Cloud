# TPRM Metrics (US-Grade Operational Dashboard)

## Coverage & timeliness
- % vendors with tier assigned
- % High/Critical vendors reassessed on time
- Average time from intake → decision (days)

## Risk posture
- Count of High vendors / Critical vendors
- Count of vendors with privileged/VPN access
- Count of vendors processing Restricted data

## Evidence quality
- % evidence items current (< 12 months) for High/Critical
- Evidence confidence distribution (A/B/C)
- Most common evidence gaps (top 5)

## Remediation performance
- Average remediation closure time (days)
- # overdue remediation items (High/Critical)
- Repeated gaps by vendor type (e.g., MSP, SaaS)

## Exceptions / risk acceptance
- # active exceptions
- Average exception duration
- # exceptions beyond expiry (should be 0)

## Monitoring triggers
- # incidents triggering reassessment
- # major change events (new access/data/subprocessors)
