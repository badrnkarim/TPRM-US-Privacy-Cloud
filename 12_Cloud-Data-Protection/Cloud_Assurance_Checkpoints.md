# Cloud Data Protection Assurance (TPRM Lens)
## Core checkpoints
- Shared responsibility clarity (who secures what)
- Encryption in transit + at rest
- Authentication (SSO/MFA) and privileged access controls
- API governance (scopes, rotation, allowlisting, deprovisioning)
- Logging + retention (access/admin/security events)
- Incident response + forensics support expectations
