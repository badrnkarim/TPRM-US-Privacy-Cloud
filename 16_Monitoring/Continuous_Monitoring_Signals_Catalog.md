# Continuous Monitoring Signals (Catalog)
- SOC2/ISO expiry and renewal status
- Public breach disclosures / advisories
- Subprocessor changes
- Major product/security changes
- New privileged access requests
- Repeated remediation slippage
- Domain/cert anomalies (where relevant)
