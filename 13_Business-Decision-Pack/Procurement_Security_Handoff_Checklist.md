# Procurement + Security Handoff Checklist

## Before contract signature
- Tier assigned (Low/Med/High/Critical)
- Evidence received and reviewed (EV-INDEX updated)
- DPA/security annex checklist completed (privacy + security clauses)
- Subprocessor disclosure reviewed
- Incident/breach notification clause confirmed (time-bound)
- Retention + deletion/return clause confirmed
- DSAR support clause confirmed (if applicable)
- Approval authority satisfied (matrix)

## Before go-live
- SSO/MFA configured
- API token governance defined (scopes + rotation)
- Logging/export path defined (SIEM if High/Critical)
- Remediation plan created for conditional items
- Monitoring cadence set

## Ongoing
- Reassessment date scheduled
- Contract renewal trigger includes re-evidence for High/Critical
