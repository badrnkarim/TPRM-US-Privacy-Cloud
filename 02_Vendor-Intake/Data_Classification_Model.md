# Data Classification (4-Level Model)

## Public
Approved for public release. No harm if disclosed.

## Internal
Internal business information. Limited harm if disclosed.

## Confidential
Business-sensitive or customer data. Disclosure can cause material harm.

## Restricted
Highest sensitivity: regulated personal data, credentials/secrets, keys, privileged logs, or data whose disclosure causes severe harm.

## Rules of use (TPRM)
- Restricted data requires High/Critical tier controls by default.
- Any vendor with privileged access to systems handling Restricted data is High/Critical by default.
